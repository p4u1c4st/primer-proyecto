Etiquettas de html

Las etiquetas HTML se usan para estructurar el contenido de una página web, indicando al navegador cómo mostrar elementos como párrafos,
títulos, imágenes, enlaces y tablas. Funcionan como marcadores que vienen en pares de apertura (tag) y cierre (/tag), aunque algunas, como
para un salto de línea, son autocontenidas. Se utilizan para dar formato y sentido al texto, organizar el contenido de manera clara y para mejorar
la accesibilidad y el SEO (optimización para motores de búsqueda).

Etiqueta tablas

La etiqueta table en HTML es la encargada de definir y estructurar datos en forma de tabla (filas y columnas). Se usa para organizar información
tabular como horarios o planes de precios. Los elementos más importantes dentro de una tabla son tr para definir filas, th para los encabezados de
las columnas, y td para las celdas de datos. Además, se utiliza caption para proporcionar un título y una descripción concisa del contenido de la tabla.
